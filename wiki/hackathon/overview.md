## 3. 참가 자격 (Eligibility)

### 가능
- 거주지 기준 성년 이상 개인
- 개인들로 구성된 팀 (대표자 1인 지정 필수)
- 법인/조직

### 불가능
- Qwen Cloud 가입 미지원 국가/제재 대상 지역 거주자·법인
- 주최/운영 관련자와 그 가족, 심사위원 및 소속사, 이해충돌 대상

## 4. 참가 절차 (How to Enter)

1. Devpost 등록 (~2분)
2. Qwen Cloud 가입 + 무료 크레딧($40 바우처) 신청 — https://www.qwencloud.com/challenge/hackathon/voucher-application (~5분)
3. Qwen Cloud Discord 참여 — https://discord.gg/cDEHSV4Qqj (~2분)
4. 트랙 선택 → Resources의 샘플/레퍼런스 아키텍처 확인 → 빌드 시작

- API Base URL: `https://dashscope-intl.aliyuncs.com/compatible-mode/v1` (**OpenAI 호환** — OpenAI SDK 사용 가능)
- $40 크레딧 초과분은 자비 부담

## 5. 프로젝트 요건 (Project Requirements)

- **Qwen Cloud의 Qwen 모델 사용** + 5개 트랙 중 최소 1개에 해당 → [tracks.md](tracks.md)
- **기능성**: 의도한 플랫폼에서 설치·실행 가능해야 하며, 영상/설명과 실제 동작 일치
- **신규성**: 해커톤 기간 중 새로 만들었거나, 기존 프로젝트라면 **기간 내 유의미한 업데이트** + 그 내역 설명 필수
- **서드파티**: SDK/API/데이터 사용 시 라이선스 준수
- **소유권**: 참가자 본인의 원저작물 (오픈소스 컴포넌트는 라이선스 준수 하에 허용, 이 위에 기능을 확장·구축해야 함)

## 6. 상금 구조 (Prizes)

| 상 | 수량 | 내역 |
|---|---|---|
| 트랙별 Grand Prize (5개 트랙) | 각 1팀 | **$7,000 현금 + $3,000 크레딧** + 블로그 피처링 + 스웨그 + 앰배서더 기회 |
| Top 10 Honorable Mention | 10팀 | $500 현금 + $500 크레딧 |
| Top 10 Blog Post Award | 10팀 | $500 현금 + $500 크레딧 |

- 한 프로젝트는 Grand Prize 1개 + Blog Post Prize 1개까지만 수상 가능
- 복수 제출 가능하나 각 제출은 상호 실질적으로 달라야 함
- 우수팀은 **AI Catalyst 프로그램** 초청

## 7. 지식재산권 (IP)

- 제출물의 IP는 참가자 소유 유지
- 주최 측에 심사·홍보 목적의 비독점 라이선스 부여 (해커톤 기간 + 3년)
- 리포는 **공개 + 오픈소스 라이선스 필수** (About 섹션에서 감지 가능해야 함)

## 8. 리스크·주의사항

- 제출 기간 종료 후 수정 불가 (침해 콘텐츠 제거 등 예외만 허용)
- 심사단은 텍스트/영상만으로 심사할 수 있음 → **영상·문서 품질이 사실상 승부처**
- 3분 초과 영상은 심사위원이 안 볼 수 있음
- 모든 제출물은 영어 (비영어 시 영어 번역 첨부)
- 테스트 접근 제공 필수 (비공개 사이트면 로그인 크리덴셜 제공)
